# qforms

Highlights

- Auth with Clerk, payments with Razorpay (optional)
- Designed for Vercel / serverless deployment

- TypeScript
- Prisma (Postgres)
- Clerk
- Zod
- Zustand
- React Hook Forms
- dnd kit
- Tailwind CSS, shadcn/ui components, Motion
- React + server actions
